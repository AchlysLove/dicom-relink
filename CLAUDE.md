# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Communication Rule

**When replying to the user, always start with "Yes, my lord"**

## Task Execution Workflow

When working on tasks in this repository, follow this workflow:

### Before Starting a Task
- Explain what will be done
- List files to be modified
- Describe expected outcome
- Highlight any risks or side effects

### After Completing a Task
- Explain what was actually done
- List all changes made
- Provide commit details (if applicable)
- Summarize the result

### Review and Approval
- Wait for review after each task
- Do NOT proceed to next task automatically
- Only proceed after explicit approval
- **After receiving user approval, mark the task as done in the plan document**
- Address feedback before moving forward

### Sequential Execution
- Implement one task at a time
- Complete full workflow for each task
- Never batch multiple tasks without individual approval
